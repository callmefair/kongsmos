변수는 <u>데이터를 저장해주는 역할</u>
일일이 반복되는 내용을 바꿔야하는데 매번 바꿀 수는 없는 노릇이지.
```python
character_name = "John"
character_age = "35"
print("There once was a man named " + character_name + ", ")
print("he was " + character_age + " yeasrs old. ")
character_name = "Mike"
print("He really liked the name " + character_name + ", ")
print("but didn't like being " + character_age + ".")
```
내용을 바꾸려면 이제 변수만 바꾸면 되겠네!
아무튼 내가 기억하고싶던건 print 할 문장 내에 변수를 넣으려면 +로 하자는거

DataType
string: text를 저장해주는 애들
number: 이걸 저장할 때는 ""가 필요가 없지
